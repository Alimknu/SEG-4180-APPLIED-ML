=== Building ResNet CNN Model ===
Architecture:
  Input: 8x8x12 board representation
  Initial Conv2D(32) -> BatchNorm -> ReLU
  2x ResidualBlock(32)
  MaxPooling2D(2x2)
  2x ResidualBlock(64)
  GlobalAveragePooling2D
  Dense(128) -> Dropout(0.5) -> Dense(10, sigmoid)

Rationale:
  - Residual connections enable training deeper networks
  - Skip connections help gradient flow during backpropagation
  - GlobalAveragePooling reduces parameters vs Flatten
  - Multiple residual blocks capture hierarchical patterns
  - Sigmoid for independent multi-label classification
